This is the installation package for the
HP StoreOpen Standalone
software version 3.0.0.

This will install the HP StoreOpen Standalone and ltfscopy utility software onto your system. Note that to use HP StoreOpen Standalone software, you should also have already installed the following packages:

  • OSXFUSE (bundled with this distribution package)
  • Refer to the HP Data Agile BURA Compatibility Matrix for compatibility of versions of OSXFUSE with corresponding Mac OS X versions
  • icu version 50.1.2 (attached to this distribution package)

This installer will not allow you to proceed until you have installed these two packages.
